# datetime overlay

A browser source overlay that displays the date and time in the following format:

    4:20am 25 Jan 2024

Recommended browser source settings:

| Field      | Value                              |
| ---------- | ---------------------------------- |
| URL        | https://datetime-overlay.pages.dev |
| Width      | 240                                |
| Height     | 40                                 |
| Custom CSS |                                    |
